# persona-intake

Branching, LLM-assisted intake questionnaire that generates an agent prompt-pack / docs (e.g. `SOUL.md`, `AUTONOMY.md`, `USER.md`).

## Contribution model
We accept PRs for:
- `question-bank/` (questions + branching rules)
- `templates/` (document templates)
- docs + examples

We do **not** auto-run contributor code. PRs are reviewed manually.

## Status
Early bootstrap. Expect schema churn.
